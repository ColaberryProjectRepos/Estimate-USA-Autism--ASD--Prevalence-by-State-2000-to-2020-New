Create Dax Measure Tables 13


Steps
 Select the Master Data Table under the fields pane by clicking on it
 From the Power BI desktop ribbon, Home tab click on New measure
 The formula bar appears at the top of the report canvas.
 Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select   Commit (checkmark icon) in the formula bar.  
 After creating each measure choose the necessary format and decimal for each value (measure)
 Save

ASD Pop = SUM ('Master Data'[Autistic Population])

Total Population = SUM ('Master Data'[Population])

ASD% = FORMAT (Divide([ASD Pop],[Total Population],0),"0.00%")

Population YoY% =
 VAR CurrentYear = MAX('Master Data'[Year])
 VAR LastYear = MAX('Master Data'[Year]) - 1
 VAR CurrentYearTotal = CALCULATE(SUM('Master Data'[Population]), 'Master Data'[Year] = CurrentYear)
 VAR LastYearTotal = CALCULATE(SUM('Master Data'[Population]), 'Master Data'[Year] = LastYear)
 RETURN
 IF(ISBLANK(LastYearTotal), BLANK(), DIVIDE(CurrentYearTotal - LastYearTotal, LastYearTotal))

 Arrow ASD YoY = SWITCH(TRUE(), [ASD Pop YoY%] <0, UNICHAR(9662),[ASD Pop YoY%] =0,   UNICHAR(9666), UNICHAR(9652))

 Arrow Population YoY = SWITCH(TRUE(), [Population YoY%] <0, UNICHAR(9662),[Population YoY%] =0, UNICHAR(9666), UNICHAR(9652))

 ASD Pop YoY% =
 VAR CurrentYear = MAX('Master Data'[Year])
 VAR LastYear = MAX('Master Data'[Year]) - 1
 VAR CurrentYearTotal = CALCULATE(SUM('Master Data'[Autistic Population]), 'Master Data'[Year] =   CurrentYear)
 VAR LastYearTotal = CALCULATE(SUM('Master Data'[Autistic Population]), 'Master Data'[Year] =   LastYear)
 RETURN
 IF(ISBLANK(LastYearTotal), BLANK(), DIVIDE(CurrentYearTotal - LastYearTotal, LastYearTotal))
